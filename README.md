## Start project

```bash
cd yape
```
```bash
yarn bootstrap
```
```bash
yarn prisma migrate dev
```

## Start Apps

### Transaction app
```bash
yarn nest start
```

### AntiFraud microservice app
```bash
yarn nest start anti-fraud
```


### Test
```bash
yarn test
```
